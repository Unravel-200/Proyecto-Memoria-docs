# Proyecto Memoria — Documentación oficial

Este repositorio es la única fuente editable de documentación para Proyecto Memoria.

## Estructura

- `Produccion/`: control maestro y versiones jugables.
- `Modelos3D/`: registro de activos, dimensiones y checklist de exportación.
- `Unreal/`: arquitectura C++ + Blueprints.

Los repositorios `ProyectoMemoria-Modelos3D` y `ProyectoMemoria-Unreal` solo deben incluir un README que apunte a esta ubicación.
